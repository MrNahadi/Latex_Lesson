# Learn LaTeX in 30 Minutes

## Overview
"Learn LaTeX in 30 Minutes" is an interactive, web-based crash course designed to teach the fundamentals of LaTeX document preparation. The project features a beautifully crafted front-end interface adopting an editorial and magazine design aesthetic, combined with a comprehensive PDF guide. It demonstrates the WYSIWYM (What You See Is What You Mean) philosophy and provides an intuitive introduction for beginners.

## Features
* **Interactive Web Tutorial**: Step-by-step instructions available directly in the browser via `lesson.html`.
* **Professional Aesthetic**: Custom, production-grade styling with a responsive sidebar, elegant typography (Cormorant Garamond, Source Sans 3, JetBrains Mono), and a polished color palette.
* **Included Reference Material**: Contains the comprehensive `LaTeX_Beginners_Lesson_Preparation.pdf` for offline reading and further reference.
* **Code and Output Panels**: Provides visual side-by-side examples of LaTeX code and its compiled output.
* **Responsive Layout**: Adapts seamlessly to various screen sizes, ensuring an optimal learning experience on desktop, tablet, and mobile devices.

## Getting Started
To begin the tutorial, simply open the `lesson.html` file in any modern web browser. No local server or installation is required to view the interactive lesson. 

* Users can navigate through chapters using the sidebar or the navigation arrows at the bottom of each page.
* The lesson covers everything from basic document structure and preamble configuration to text formatting and mathematical equations.

## Project Structure
* `lesson.html`: The main interactive web application containing the tutorial content, styling, and frontend logic.
* `LaTeX_Beginners_Lesson_Preparation.pdf`: The downloadable reference document containing the full lesson material.
